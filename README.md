> Average completion time: 5-8s - 7x faster than self-implemented workflow (with
> cache enabled).

## Usage

```yml
- uses: actions/checkout@v2 # Check out the repository first.
- uses: actions-prettier
  with:
    # prettier CLI arguments.
    args: --check .
```
